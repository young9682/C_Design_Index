# 银行排队调度仿真系统

基于离散事件仿真（DES）的高保真银行网点排队调度模拟程序，使用 C 语言实现。

## 功能概述

- **分时段泊松到达**：08:00-17:00 按小时划分 9 个时段，各时段独立到达率
- **客户画像分层**：VIP（10%）、企业（15%）、老年（20%）、普通（55%），不同画像对应不同业务概率和优先级
- **5 类业务办理**：现金存取、转账汇款、开户销户、理财咨询、对公结算，耗时服从对数正态分布
- **多级优先级队列**：按预约标识 > 画像权重 > 先到先得排序
- **弹性潮汐窗口**：高峰时段自动开启/关闭潮汐窗口
- **Balking 弃号**：队列过长时客户以一定概率放弃排队
- **Jockeying 换队**：队列长度差异大时客户换到更短的队列
- **过号重排**：叫号超时未响应，客户降权后重新入队
- **异常中断**：2% 概率触发窗口故障，2 分钟后恢复
- **批量突发到达**：随机时刻一次性涌入 3-8 人
- **营业时间约束**：17:00 停止取号，20:00 强制关门

## 编译与运行

```bash
# 编译
g++ -o bank_sim bank_simulation.cpp -lm

# 运行
./bank_sim
```

## 输出文件

| 文件 | 说明 |
| :--- | :--- |
| `report.txt` | 仿真统计报告（总体统计、各画像等待时间、各窗口统计、排队与满意度、均衡度） |
| `simulation.log` | 仿真过程日志（带时间戳的事件记录） |

控制台同时输出关键事件的实时日志。

## 默认参数

### 营业时间

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| `open_time` | 0（08:00） | 开门时间 |
| `close_stop_time` | 540（17:00） | 停止取号时间 |
| `close_force_time` | 720（20:00） | 强制关门时间 |

> 当前 main() 中设置了测试参数：`close_force_time=60`，`close_stop_time=50`，仅仿真 60 分钟。

### 时段到达率 λ（人/分钟）

| 时段 | 时间 | λ |
| :--- | :--- | :--- |
| 0 | 08:00-09:00 | 8.0（早高峰） |
| 1 | 09:00-10:00 | 4.0 |
| 2 | 10:00-11:00 | 3.0 |
| 3 | 11:00-12:00 | 3.0 |
| 4 | 12:00-13:00 | 3.5 |
| 5 | 13:00-14:00 | 6.0（午后高峰） |
| 6 | 14:00-15:00 | 4.0 |
| 7 | 15:00-16:00 | 3.0 |
| 8 | 16:00-17:00 | 3.0 |

### 客户画像概率

| 画像 | 概率 | 优先级权重 |
| :--- | :--- | :--- |
| VIP | 10% | 3 |
| 企业客户 | 15% | 2 |
| 老年客户 | 20% | 2 |
| 普通客户 | 55% | 1 |

### 业务耗时参数（对数正态分布）

| 业务 | mu | sigma | 典型耗时 |
| :--- | :--- | :--- | :--- |
| 现金存取 | 2.0 | 0.4 | ~7.4 分钟 |
| 转账汇款 | 2.5 | 0.5 | ~12.2 分钟 |
| 开户销户 | 3.0 | 0.6 | ~20.1 分钟 |
| 理财咨询 | 3.5 | 0.5 | ~33.1 分钟 |
| 对公结算 | 4.0 | 0.6 | ~54.6 分钟 |

> 实际耗时 = 基准时长 x 柜员熟练度系数 k（0.8~1.5）

### 窗口配置

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| `window_count` | 5 | 总窗口数 |
| `tidal_window_count` | 2 | 潮汐窗口数 |
| `tidal_start` | 0 | 潮汐开启时间（08:00） |
| `tidal_end` | 540 | 潮汐关闭时间（17:00） |

### 调度参数

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| `balking_threshold` | 10 | 弃号队列长度阈值 |
| `balking_probability` | 0.3 | 弃号概率 |
| `jockey_diff` | 2 | 换队长度差阈值 |
| `jockey_probability` | 0.25 | 换队概率 |
| `error_probability` | 0.02 | 异常中断概率 |
| `no_response_timeout` | 3.0 分钟 | 过号超时时间 |

### 批量突发

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| `bulk_time_start` | 60（09:00） | 突发最早触发时间 |
| `bulk_time_end` | 480（16:00） | 突发最晚触发时间 |
| `bulk_count_min` | 3 | 突发人数下限 |
| `bulk_count_max` | 8 | 突发人数上限 |

## 事件类型

| 枚举值 | 说明 |
| :--- | :--- |
| `ARRIVAL` | 单个客户到达 |
| `BULK_ARRIVAL` | 批量突发到达 |
| `START_SERVICE` | 开始办理 |
| `END_SERVICE` | 办理结束 |
| `WINDOW_SWITCH` | 窗口类型切换 |
| `NO_RESPONSE_TIMEOUT` | 过号超时 |
| `SHUTDOWN` | 营业结束 |

## 代码结构

单文件架构，按注释分区：

```
bank_simulation.cpp
  ├── 类型定义区（枚举 + 结构体）
  ├── P1 分区：随机引擎（MT19937）+ 配置 + 工具函数
  ├── P2 分区：事件表（最小堆）+ 优先队列 + 窗口操作 + 仿真引擎
  ├── P3 分区：业务逻辑（到达/服务/调度）
  ├── P4 分区：统计收集器 + 日志 + 报告生成
  └── main() 主程序
```
