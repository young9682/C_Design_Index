# 智能共享充电宝租借柜系统 - 最终需求文档

## 一、系统配置

程序启动时由用户输入：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 槽位数量 N | 充电宝总数 | 6 |
| 费率 | 每半小时价格（元） | 1.5 |

## 二、数据结构

### 充电宝状态（State）

| 枚举值 | 含义 | 能否被租借 |
|--------|------|-----------|
| `STATE_AVAILABLE` | 空闲，在槽位内 | 能 |
| `STATE_RENTED` | 租借中，已被借出 | 不能 |
| `STATE_CHARGING` | 充电中，电量不足正在充电 | 不能 |

### 充电宝属性（PowerBank）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 编号（1 ~ N） |
| state | State | 当前状态 |
| battery | float | 电量（0-100%） |
| rental_start | time_t | 租借开始时间（仅 RENTED 时有效） |

### 充电柜（Cabinet）

| 字段 | 类型 | 说明 |
|------|------|------|
| banks | PowerBank* | 充电宝数组 |
| total | int | 槽位数量 |
| price | float | 每半小时费率（元） |

## 三、初始化

- 所有 N 个充电宝：状态 = 空闲，电量 = 100%
- 编号从 1 开始（1 ~ N）

## 四、用户操作

### 4.1 租借

1. 查找所有 `STATE_AVAILABLE` 的充电宝
2. 若无空闲 → 提示"暂无可用电宝"，返回
3. 若有 → 随机选择一个
4. 状态改为 `STATE_RENTED`
5. 记录 `rental_start = 当前时间`
6. 提示"租借成功，充电宝编号 X"

### 4.2 归还

1. 用户输入充电宝编号
2. 校验：
   - 编号是否存在（1 ~ N）
   - 状态是否为 `STATE_RENTED`
3. 若校验失败 → 提示错误信息，返回
4. 若校验通过：
   - 计算租借时长和费用
   - 随机生成归还电量（20% ~ 90%）
   - 根据电量决定下一状态：
     - 电量 ≥ 50% → 状态变 `STATE_AVAILABLE`，开始充电
     - 电量 < 50% → 状态变 `STATE_CHARGING`，开始充电
   - 显示归还结果（编号、时长、费用、后续状态）

## 五、充电规则

### 5.1 充电速度

| 条件 | 充电速度 |
|------|---------|
| 电量 < 80% | 每秒 +0.25% |
| 电量 ≥ 80% | 每秒 +0.20% |

### 5.2 充电上限

- 电量最高 100%，不可超过

### 5.3 状态转换

- 充电中，电量达到 100% → 状态自动变为 `STATE_AVAILABLE`

## 六、计费规则

### 6.1 计费周期

- 以**每半小时**为一个计费周期
- 不满一个周期按一个周期计算（向上取整）

### 6.2 计算公式

```
租借时长（秒）= 当前时间 - rental_start
计费周期数 = ceil(租借时长（秒）/ 1800)
费用 = 计费周期数 × 费率
```

### 6.3 示例

| 租借时长 | 计费周期数 | 费率 1.5 元 | 费率 2 元 |
|---------|-----------|------------|----------|
| 10 分钟 | 1 | 1.5 元 | 2 元 |
| 30 分钟 | 1 | 1.5 元 | 2 元 |
| 31 分钟 | 2 | 3 元 | 4 元 |
| 1 小时 | 2 | 3 元 | 4 元 |
| 1.5 小时 | 3 | 4.5 元 | 6 元 |

## 七、状态显示

实时显示所有充电宝信息：

```
===== 充电宝状态 =====
编号 | 状态     | 电量
--------------------
  1  | 空闲     | 100%
  2  | 租借中   |  --
  3  | 充电中   | 65%
  4  | 空闲     | 100%
======================
```

- 空闲/充电中：显示电量
- 租借中：电量显示 `--`（不在柜内）

## 八、主循环流程

```
程序启动
  ├─ 输入 N 和费率
  ├─ 初始化 N 个充电宝
  └─ 主循环 (while 1)
       ├─ update_charging()  # 更新所有充电中状态的电量
       ├─ display_status()   # 显示所有充电宝状态
       ├─ 显示菜单 [1.租借 2.归还 0.退出]
       ├─ 处理用户输入
       └─ sleep(1)           # 模拟 1 秒时间流逝
```

## 九、菜单设计

```
欢迎进入共享充电宝租借柜系统！
1. 租借充电宝
2. 归还充电宝
0. 退出系统
请选择操作：
```

## 十、输入校验

| 场景 | 处理 |
|------|------|
| 租借时无空闲充电宝 | 提示"暂无可用电宝" |
| 归还时编号不存在 | 提示"编号无效，请重新输入" |
| 归还时状态非租借中 | 提示"该充电宝未被租借" |
| 菜单输入无效选项 | 提示"无效选项，请重新选择" |
