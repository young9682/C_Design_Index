# 实验四：简易自然语言文本单词与情感统计
## 一、实验目的
掌握 C 语言字符串处理、字符遍历、大小写转换、多分隔符分词等基础文本处理算法；
学会利用 `strtok` 实现多标点、多空格混合场景下的简易分词，理解基础文本分词逻辑；
实现文本单词统计、字符长度计算、关键词匹配功能，初步体验极简自然语言处理流程；
结合情感词库完成文本情感判别，建立“文本输入—分词清洗—特征统计—结果输出”完整处理流程。
## 二、涉及知识点
字符串操作、分词算法，字符处理及文本归一化。
## 三、问题描述
编写 C 语言程序，实现带标点英文短句的文本分析，功能要求如下：
### 输入处理：读取一行包含英文、空格、逗号、句号、问号、感叹号的英文句子；
### 文本清洗分词：
1. 支持空格` `、逗号`,`、句号`.`、问号`?`、感叹号`!`作为分词分隔符，同时要考虑输入不规范的情况；
2. 自动将所有单词统一转为小写，消除大小写对词汇匹配的影响；
3. 过滤分割后产生的空字符串，避免统计数据失真；
### 基础文本统计：
1. 统计有效单词总个数；
2. 计算所有单词的平均字符长度，保留两位小数；
### 情感词汇匹配：
　内置正向词汇：`good`、`nice`、`happy`等；内置负向词汇：`bad`、`sad`、`worse`等；统计文本中正负词汇各自出现次数；
情感倾向判断并输出：
- 积极词数量 > 消极词：输出偏积极；
- 消极词数量 > 积极词：输出偏消极；
- 两者数量相等：输出中性；
规范打印全部统计数据与情感结论。
### 四、程序功能
- 输入：控制台手动输入一行英文短句，可混合字母、空格、`,` `.` `?` `!` 等标点；
- 输出：依次打印单词总数、单词平均长度、积极词数量、消极词数量、文本情感倾向。
> 输入：
> Happy day, I feel good but the weather is bad and sad!  
输出：
```bash
===== 文本分析结果 =====
单词总数量：12
单词平均长度：3.57
积极词汇出现次数：2
消极词汇出现次数：2
文本情感倾向：中性
```
